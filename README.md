# minicelslicer

## A quick and cheap traditional animation method


Instructions:

Download and install imagemagick https://imagemagick.org/

Download MINI CELS BLANK.pdf

Print and draw your cels

Scan it back in with flatbed scanner- should measure W-2552px H-3508px

Put each image in its own folder, rename them all 'img1.jpg', type cmd in the file location and paste the .txt command

Your slices will come out x4 for a looping 10-frame animation. You can paste just the first 10 lines if you just want the 10.
